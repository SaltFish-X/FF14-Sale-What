# [FF14-SaleWhat](https://salewhat14.netlify.app/)

基于universalis的API编写了一套操作更加方便的FF14物价查询系统，拥有以下功能
1. 支持国服和国际服的在售物品价格和已售物品价格，便于判断当前什么东西卖的好
2. 收藏夹功能便于一次性进行大量查询
3. 历史成交记录，便于知道是谁在什么服务器进行了购买
